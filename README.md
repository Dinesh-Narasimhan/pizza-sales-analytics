## Business Context
This project analyzes transactional sales data for a mid-to-large food retail enterprise to evaluate product performance, revenue drivers, and customer ordering behavior.

## Objective
- Identify high-revenue product categories and sizes
- Analyze time-based sales patterns
- Provide executive-ready dashboards and insights

## Dataset
- 48,620 transaction records
- 21,350 unique orders
- One full fiscal year of sales data

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (MySQL)
- Tableau (Executive Dashboard)

## Key Insights
- Total Revenue: $817,860
- Large pizzas contribute 46% of total revenue
- Fridays generate the highest daily revenue
- Classic category leads in volume and revenue

## Deliverables
- Cleaned dataset
- SQL analysis scripts
- Python notebook
- Interactive Tableau dashboard
- Executive project report

## Business Impact
The analysis enables leadership to optimize pricing, staffing, and promotional strategies, supporting data-driven revenue growth.
